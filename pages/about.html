About the Cat Map

The Arnold Cat Map is a part of a larger class of dynamical systems that are chaotic, area-preserving maps on the torus, called Hyperbolic Toral Automorphisms.


The example we use was first described by Vladimir Arnold in the 1960s using an image of a CAT (Continuous Automorphism Torus) hence the name!

How the transform works

Each step of the Cat Map happens in two or three stages:

  • If the determinant is −1, there will be a flip across the diagonal.
  • The matrix shears the grid, taking points outside the original square.
  • The modulo folds everything back in.

So what does it actually do?

The Cat Map takes every single square and moves it to a new position according to a mathematical rule. Do this once and the image looks scrambled. Do it again and it scrambles further. Keep going and eventually... it comes all the way back to the original. Every single pixel in exactly the right place.

This is the key idea, the scrambling is not random. It follows a precise rule, which means it can always be undone. That's what makes it useful for ...dun, dun, duuun... encryption .

(x′, y′) = ( a₁₁x + a₁₂y , a₂₁x + a₂₂y ) mod N

This formula describes where each pixel at position (x, y) moves to. N by N is the number of pixels of the image. You can change the matrix to get a new CAT map and a completely different scrambling pattern.

Which matrices are best to use?

You can choose any 2×2 matrix, but not all matrices are equally good for scrambling. In this case, we want the map to be area preserving so that the matrix should have determinant +1 or -1 and none of its eigenvalues should have length 1.

What is a determinant?

The determinant is a single number that describes what a matrix does to area. For a 2×2 matrix:

det(A) = a₁₁·a₂₂ − a₁₂·a₂₁

Think of it this way — if you draw a square on the grid and apply the matrix, the determinant tells you how much the area of that square changes:

  • det = 1 means the area stays exactly the same — the transformation is area-preserving.
  • det = −1 means the area stays the same but the shape gets flipped — reflected across the diagonal.
  • det = 2 means the area doubles. det = 0.5 means it halves.
  • det = 0 means the shape collapses completely — the matrix is not invertible and the image cannot be recovered.

For the Arnold Cat Map we need the determinant to be exactly ±1. This guarantees that no pixel is lost or duplicated — every pixel moves to exactly one new position, and the transformation can always be undone. This is why the site shows a warning when you enter a matrix with a different determinant.

What are eigenvalues?

Eigenvalues describe the directions in which a matrix purely stretches or compresses, without rotating. For a 2×2 matrix A, the eigenvalues λ are found by solving the characteristic equation:

λ² − (a₁₁ + a₂₂)·λ + det(A) = 0

Using the quadratic formula this gives:

λ = ( (a₁₁ + a₂₂) ± √( (a₁₁ + a₂₂)² − 4·det(A) ) ) / 2

For the Cat Map to produce chaotic scrambling, the eigenvalues need to behave in a specific way:

  • One eigenvalue must be greater than 1 in absolute value, this is the stretching direction.
  • One eigenvalue must be less than 1 in absolute value, this is the compressing direction.
  • If both eigenvalues are greater than 1, everything expands without bound, no chaos.
  • If both are less than 1, everything collapses, no chaos.
  • If either eigenvalue equals exactly 1, there is no stretching in that direction,the scrambling is weak or non-existent.

For the classic matrix A = [[1, 1], [1, 0]], the eigenvalues are (1 ± √5) / 2 — approximately 1.618 and −0.618. These are the golden ratio and its conjugate. One is greater than 1, one is less than 1 in absolute value, which is exactly what produces the strong chaotic scrambling the Cat Map is known for.

What is mod?

The mod part of the formula is short for modulo — it's the remainder after division. For example, 13 mod 10 = 3, because 13 ÷ 10 leaves a remainder of 3. You already use modulo every day when you read a clock — 14:00 is just 14 mod 12 = 2 o'clock.

In the Cat Map, mod N keeps all the pixel positions inside the image. Without it, pixels would fly off the edges of the grid. With it, they wrap around, like a game where walking off the right edge brings you back on the left. Mathematicians call this a torus, the same shape as a doughnut.

What is encryption?

You've probably heard of encryption before, it's how your messages, passwords, and photos are kept private online. Encryption takes something readable and scrambles it so that only someone with the right key can unscramble it.

Most encryption works with text or numbers. But what about images? Can we scramble a picture the same way? The Arnold Cat Map is one answer to that question and it uses a surprising amount of interesting mathematics to do it.

How would you use this to encrypt an image?

Here's a simple encryption scheme using the Cat Map:

  1. Choose a matrix A — this is your secret key. Keep it private.
  2. Apply the Cat Map to your image a chosen number of times — say, 7 steps. This is the encrypted image. It looks like noise.
  3. Send the scrambled image to your friend. Without the key, they can't recover it.
  4. Your friend applies the same Cat Map the remaining steps until the period is complete — and the original image reappears.

In practice, real image encryption uses more sophisticated techniques, but the Cat Map is used as a component in several modern encryption algorithms because of its strong scrambling properties.

Why does it always come back?

Here's the surprising part. No matter what image you start with, and no matter what matrix you choose, the image always returns to its exact original state after enough steps. Always. This is guaranteed by mathematics.

The reason is that the image is a finite grid — for example 15×15 = 225 pixels. Each pixel can only be in one of 225 positions. So there are only a limited number of ways to arrange them. Eventually the map has to repeat itself, and because it follows a strict rule, it must come back to the very beginning.

The number of steps it takes to come back is called the period. For the classic Cat Map on a 15×15 grid, the period is 40. On a 16×16 grid it is only 12. The period can vary wildly depending on the grid size — some sizes take hundreds of steps to return.

Who invented it?

The map was introduced by the Russian mathematician Vladimir Arnold in the 1960s. He demonstrated it using a picture of a cat — which is where the name comes from. Arnold made major contributions to mathematics and physics throughout his career, and the Cat Map remains one of the most well-known examples of a chaotic dynamical system that is simple enough to understand but rich enough to be genuinely interesting.
